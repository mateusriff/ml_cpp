A Machine Learning Library in C++

By Mateus Riff • Feb. 2025

C++ Machine Learning Library

This project is a simple yet effective Machine Learning library implemented in C++ using Object-Oriented Programming (OOP) principles. It provides an abstraction layer for regression models, allowing users to easily train and use them for predictive tasks.

OOP Principles in Action

The library follows OOP principles such as encapsulation, inheritance, abstraction, and polymorphism. The core class, Regression, is an abstract base class that defines shared attributes and methods for its derived classes: LinearRegression and LogisticRegression.

  • Encapsulation: The base class holds common data members such as model parameters and learning functions.
  • Inheritance: The concrete regression classes inherit from Regression, ensuring code reuse and structure.
  • Abstraction: Clients only interact with the interface without needing to know implementation details.
  • Polymorphism: Each derived class implements its own version of cost computation and prediction methods.

Usage

The library can be built using CMake and easily integrated into other C++ projects. Clients can create, train, and use models with just a few lines of code: 

#include <vector>
#include "ml_lib/ml.h"

int main() {
    ml::LinearRegression model;
    model.fit(X, y, 0.01f, 1000);
    float prediction = model.predict({5.0f, 6.0f});
    return 0;
}

Evaluation

The library's models were tested against scikit-learn implementations. The ml_cpp Linear Regression model achieved a lower Mean Squared Error (MSE) than the equivalent in scikit-learn, demonstrating its efficiency. However, the Logistic Regression model performed worse, highlighting areas for potential improvement.
